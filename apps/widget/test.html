How to Embed in Your Site

Once you've tested the widget and it's working, you can embed it on your own website:

Method 1: Script Tag (Easiest)

<script src="http://localhost:5173/dist/widget.umd.js"
        data-reactly-api-key="YOUR_API_KEY"
        data-reactly-project-id="YOUR_PROJECT_ID"
        data-position="bottom-right">
</script>

Method 2: NPM Package

import { initFeedbackWidget } from '@reactly/feedback-widget';

initFeedbackWidget({
  apiKey: 'YOUR_API_KEY',
  projectId: 'YOUR_PROJECT_ID',
  apiUrl: 'http://localhost:3001/api',
  position: 'bottom-right',
  theme: {
    primaryColor: '#3b82f6',
  }
});
